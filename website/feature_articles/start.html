OsmAnd Features

When you launch OsmAnd for the first time, all the parameters will be set to default settings. You can configure them by changing features in the menu. As you launch the app, you'll see the dashboard - a starting screen with different cards containing information about different groups of settings or files. The dashboard shows your Favourites (favourite places), your GPX tracks list, the plugins information, etc.

Tap the Settings button in the right upper corner to see the app use options. You'll be able to change the parameters of your screen, the map style, map layers, etc. by tapping Configure screen button.

Also, you have to download a World basemap. It is a basic map that will later help you view the maps of the regions and countries. The free version of OsmAnd allows 10 free downloads.

The OsmAnd+ version allows unlimited downloads and map updates with all the new details. The application uses Open Street Map resources - the maps created by contributors from all over the world. People add new items to the map, refine the details and delete the points of interest (POI) that are no longer there. Thus, the updates will provide the freshest information available. To download the map of the region you need, go to Settings - Manage map files - All downloads and choose the one you need.

Setting up OsmAnd for your region

To set app language, please go to Settings - General - Display language. Now all your menus will be in the chosen language.

To change the map names language, please go to Configure map - Map language. You can set it to Local names and the map names will be displayed in the language of the region or you can choose one language for all names.

Driving region

This parameter will affect the units of measure used on the map, the appearance of the speed limit sign shown when navigating, and right-hand or left-hand driving.

Units of measure: Select kilometres/metres, miles/feet, or miles/yards as desired. This affects all distance and speed displays and announcements. You can also change this later in Settings - General Settings - Driving region and Settings - General Settings - Units of measure.

Setting up navigation

To choose navigation language, go to Settings - Navigation - Voice guidance. Please prefer a synthesized (tts) voice over a pre-recorded voice, if possible for your language. Synthesized voices are more flexible and can pronounce distances and times in finer steps, they also pronounce street names and your waypoint names, which pre-recorded voices can not do.

If you select a tts voice, you need to have a respective text-to-speech engine in your Android operating system. These are partly included in Android or can be installed separately. Please go to your Android (device) settings, find Language and keyboard / Text-to-speech output or similar. Select or install an engine, then also install the language support for the language you need, this may be an extra setting or small download.

Please test ('Listen to an example' or similar test button provided) if your Android tts is working ok. Then you may also test if your OsmAnd voice prompts have been selected properly by going to Settings / General / OsmAnd development / Test voice prompts in OsmAnd. (OsmAnd debugging plugin must be activated to see the setting).
